## การคำนวณยอดผ่อนเบื้องต้น (สูตร)

### สูตรคำนวณ
1. ยอดจัดไฟแนนซ์ = ราคารถ - เงินดาวน์
2. ดอกเบี้ยรวม = ยอดจัด x อัตราดอกเบี้ยต่อปี x จำนวนปี
3. ยอดผ่อนรวม = ยอดจัด + ดอกเบี้ยรวม
4. ผ่อนเดือนละ = ยอดผ่อนรวม / จำนวนงวด

### ตัวอย่างการคำนวณอย่างง่าย
- ราคารถ: 600,000 บาท
- ดาวน์ 20%: 120,000 บาท
- ยอดจัด: 480,000 บาท
- ดอกเบี้ย 2.79% x 5 ปี = 13.95%
- ดอกเบี้ยรวม: 480,000 x 13.95% = 66,960 บาท
- ผ่อน 60 งวด: (480,000 + 66,960) / 60 = 9,116 บาท/เดือน
