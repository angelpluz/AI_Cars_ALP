# รุ่นรถโตโยต้าที่จำหน่ายที่โตโยต้า ธนบุรี

ศูนย์ผู้จำหน่ายโตโยต้า ธนบุรี จำหน่ายรถยนต์โตโยต้าทุกรุ่นที่วางจำหน่ายในประเทศไทย

---

### Toyota bZ4X
ประเภท: รถอเนกประสงค์ไฟฟ้า 100% (Electric SUV)
รุ่นย่อยและราคา:
1. **bZ4X FWD** - 1,529,000 บาท
2. **bZ4X AWD** - 1,649,000 บาท

อุปกรณ์ภายนอก (ทั้ง 2 รุ่น):
- ไฟหน้า LED, ไฟท้าย LED
- หลังคาพาโนรามิค
- ระบบไฟหน้าอัตโนมัติ
- กระจังหน้าบนสีเดียวกับตัวรถ / กระจังหน้าล่างสีดำ
- เสาอากาศครีบฉลาม, ที่ปัดน้ำฝนอัตโนมัติ
- กระจกมองข้างสีดำพร้อมไฟเลี้ยว LED ปรับและพับไฟฟ้าอัตโนมัติ พร้อมระบบบันทึกตำแหน่งและ Reverse Link
- กระจก UV Cut รอบคัน, มีบังโคลนซุ้มล้อหน้า-หลัง, ระบบปรับระดับไฟหน้าสูง-ต่ำ
- สปอยเลอร์หลัง และไฟส่องสว่างที่กระจกมองข้าง

อุปกรณ์ภายในและความสะดวก:
- เบาะหนังสังเคราะห์
- เบาะคนขับปรับไฟฟ้า 8 ทิศทางพร้อมระบบบันทึกตำแหน่ง
- เบาะผู้โดยสารหน้าปรับไฟฟ้า 8 ทิศทาง
- สีภายในดำ/เทาอ่อน (ขึ้นกับสีภายนอก)
- หน้าจอผู้ขับขี่สี 7 นิ้ว, หน้าจอสัมผัส 14 นิ้ว, มาตรวัด Digital
- รองรับ Bluetooth / Apple CarPlay / Android Auto / T-Connect / USB
- ประตูท้ายไฟฟ้าพร้อม Kick-activated
- Smart Entry + Push Start
- ระบบปรับอากาศอัตโนมัติ 2 โซน พร้อมช่องแอร์หลัง
- กระจกมองหลังแบบดิจิทัล (Digital Rear View Monitor)
- ไฟ Ambient ปรับได้ 64 เฉดสี
- เบรกมือไฟฟ้า EPB พร้อม Auto Brake Hold
- ระบบกรองฝุ่น PM2.5 และ nanoe X
- Paddle สำหรับควบคุมแรงหน่วงเบรก, Wireless Charger 2 ตำแหน่ง, ระบบช่วยจอด IPA
- ลำโพง: AWD เป็น JBL 9 ตำแหน่ง / FWD เป็น 6 ตำแหน่ง
- โหมดการขับขี่: AWD = ECO / NORMAL / X-Mode, FWD = ECO / NORMAL

ระบบความปลอดภัย:
- ถุงลม SRS คู่หน้า / ด้านข้างคู่หน้า / ม่านด้านข้าง / ตรงกลางด้านหน้า / หัวเข่าคนขับ
- BSM + RCTA + Safe Exit Assist
- Dynamic Radar Cruise Control แบบ All-Speed พร้อมระบบลดความเร็วอัตโนมัติขณะเข้าโค้ง
- PKSB ด้านหน้าและด้านหลัง
- กล้องมองรอบคัน PVM
- ชุดระบบมาตรฐาน: ABS/BA/EBD/VSC/TRC/HAC/PCS/LDA/LTA/AHB/ESS
- ISOFIX, TPMS, เซ็นเซอร์กะระยะหน้า-หลัง
- ระบบควบคุมความเร็วขณะลงทางลาดชัน (DAC): มีใน AWD, ไม่มีใน FWD

มิติรถ:
- ขนาดตัวถัง (ยาว x กว้าง x สูง): 4,690 x 1,860 x 1,650 มม.
- ฐานล้อ 2,850 มม.
- จำนวนที่นั่ง 5 ที่นั่ง
- ช่วงล้อหน้า/หลัง 1,600 / 1,610 มม.
- ระยะต่ำสุดจากพื้น: AWD 200 มม. / FWD 201 มม.
- รัศมีวงเลี้ยวแคบสุด 5.6 ม.

ระบบขับเคลื่อนและช่วงล่าง:
- AWD: ขับเคลื่อน 4 ล้อ (All-Wheel-Drive)
- FWD: ขับเคลื่อนล้อหน้า
- ระบบส่งกำลัง eAxle
- ช่วงล่างหน้า MacPherson Strut + เหล็กกันโคลง
- ช่วงล่างหลัง Double Wishbone + เหล็กกันโคลง
- ล้ออัลลอย 20 นิ้ว ยาง 235/50 R20
- เบรกหน้า/หลัง ดิสก์เบรก
- พวงมาลัยพาวเวอร์ไฟฟ้า EPS

มอเตอร์ไฟฟ้าและแบตเตอรี่:
- มอเตอร์หน้า: 167 kW (227 PS / 224 HP), แรงบิด 269 นิวตันเมตร (ทั้ง AWD/FWD)
- มอเตอร์หลัง (เฉพาะ AWD): 88 kW (120 PS / 118 HP), แรงบิด 170 นิวตันเมตร
- กำลังรวม: AWD 252 kW (343 PS / 338 HP) / FWD 165 kW (224 PS / 221 HP)
- มอเตอร์ชนิด Permanent Magnet Synchronous Motor
- แบตเตอรี่ลิเธียมไอออน แรงดัน 384.8 โวลต์
- จำนวน 26 โมดูล (104 เซลล์), ความจุ 73.11 kWh
- ระยะทางวิ่งสูงสุด (NEDC): AWD 570 กม. / FWD 600 กม.
- รองรับชาร์จ AC Type 2 สูงสุด 22 kW และ DC CCS2 สูงสุด 150 kW

หมายเหตุ:
- ระยะทางวิ่งสูงสุดเป็นตัวเลขมาตรฐาน NEDC ซึ่งมักสูงกว่าการใช้งานจริง
